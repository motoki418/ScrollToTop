# ScrollToTop
ボタンをタップ時に、指定の位置までViewをスクロールするプロジェクト

## このリポジトリの目的
ScrollViewの使い方を学習し、カスタムモディファイアを作成してViewに適用させる。

## このサンプルで学んだこと
ScrollViewReaderを使用して指定の箇所までViewをスクロールをする方法
